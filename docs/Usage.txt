Usage document for PC3

Server:
    Available to all users:
     - authenticate client
     - get scoring table
     - get a listing of problems
     - get a problem's description
     
    Also available to team clients:
     - run team solutions and add to score

    Also Available to supervisor clients:
     - override a team's completion state or score on a problem
     - get active scoring runs
     - kill a team's scoring run (presumably malfunctioning)
    
Client:
    Available to all users:
     - Login
     - Logout
     - view scores
     - view problems+descriptions
     
    Also Available to teams:
     - submit scoring run
     
    Also Available to supervisors:
     - view active scoring runs
     - request certain scoring run be killed
     - request score change for a team for a problem
     
Use cases:
    User wants to see scores
    <- client must receive score table from server
    <- server must send score table
    <- client must request score table from server
    <- user must ask client to view scores (implied in navigation)
    <- client must receive verification of username and password authentication
    <- server must authenticate username and password
    <- client must request server to authenticate username and password
    <- user must enter username and password to client
    <- client must prompt user for username and password
    <- user must load client
    <- server must send client to user
    <- user must request client from server
